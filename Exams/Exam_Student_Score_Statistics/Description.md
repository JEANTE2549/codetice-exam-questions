# สถิติคะแนนนักเรียน

**Difficulty:** Medium

## Problem

ให้เขียนโปรแกรมภาษา Python เพื่อคำนวณค่า mean, standard deviation และ mode ของคะแนนนักเรียน

คะแนนที่ถูกต้องต้องอยู่ในช่วง `0` ถึง `100` เท่านั้น หากคะแนนอยู่นอกช่วงนี้ให้ตัดออกจากการคำนวณ

ถ้าไม่มีคะแนนที่ถูกต้องเลย ให้แสดงผลว่า `No valid score`

## Mean

Mean หรือค่าเฉลี่ย คำนวณจากผลรวมของคะแนนที่ถูกต้องทั้งหมด หารด้วยจำนวนคะแนนที่ถูกต้อง

```text
mean = sum(valid_scores) / n
```

โดย `n` คือจำนวนคะแนนที่ถูกต้อง

## Standard Deviation

ให้ใช้สูตร sample standard deviation:

```text
standard_deviation = sqrt(sum((score - mean)^2) / (n - 1))
```

ถ้ามีคะแนนที่ถูกต้องเพียง `1` ค่า ให้ถือว่า standard deviation เป็น `0.00`

## Mode

Mode คือคะแนนที่พบมากที่สุดในกลุ่มคะแนนที่ถูกต้อง

ถ้ามีคะแนนหลายค่าที่มีความถี่สูงสุดเท่ากัน ให้แสดงคะแนนที่มีค่าน้อยที่สุด

## Input

บรรทัดแรกเป็นจำนวนเต็ม `N` แทนจำนวนคะแนนทั้งหมด

อีก `N` บรรทัดถัดมาเป็นจำนวนเต็ม แทนคะแนนแต่ละค่า

## Output

ถ้ามีคะแนนที่ถูกต้อง ให้แสดงผลทั้งหมด 3 บรรทัด:

```text
<mean>
<standard deviation>
<mode>
```

- `mean` ต้องแสดงทศนิยม 2 ตำแหน่ง
- `standard deviation` ต้องแสดงทศนิยม 2 ตำแหน่ง
- `mode` แสดงเป็นจำนวนเต็ม

ถ้าไม่มีคะแนนที่ถูกต้อง ให้แสดง:

```text
No valid score
```
