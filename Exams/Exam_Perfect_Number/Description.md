# Perfect Number

**Difficulty:** Medium

## Problem

ให้เขียนโปรแกรมภาษา Python เพื่อหาจำนวน perfect number ทั้งหมดตั้งแต่ `1` ถึงจำนวนที่ผู้ใช้ป้อน

Perfect number คือจำนวนเต็มบวกที่มีค่าเท่ากับผลรวมของตัวหารแท้ทั้งหมดของจำนวนนั้น โดยตัวหารแท้คือ divisor ที่หารลงตัว แต่ไม่นับตัวมันเอง

เช่น `6` เป็น perfect number เพราะตัวหารของ `6` คือ `1`, `2`, `3`, และ `6`

ตัวหารแท้ของ `6` คือ `1`, `2`, และ `3`

```text
1 + 2 + 3 = 6
```

ดังนั้น `6` จึงเป็น perfect number

## Input

จำนวนเต็มบวก `N` จำนวน 1 ค่า

## Output

แสดง perfect number ทุกจำนวนตั้งแต่ `1` ถึง `N` โดยแสดงจำนวนละ 1 บรรทัด

ถ้าไม่มี perfect number ในช่วงดังกล่าว ให้แสดง:

```text
No perfect number.
```
