# Return of Discrete

**Difficulty:** Easy

## Problem

จงเขียนโปรแกรมภาษา Python เพื่อคำนวณว่าเลขฐานสองที่กำหนดให้ มีค่าเท่ากับจำนวนใดในเลขฐานสิบ

เลขฐานสองประกอบด้วยตัวเลข `0` และ `1` เท่านั้น

ตัวอย่างเช่น `101` ในฐานสอง มีค่าเท่ากับ:

```text
1 * 2^2 + 0 * 2^1 + 1 * 2^0 = 5
```

ดังนั้น `101` มีค่าเท่ากับ `5` ในฐานสิบ

## Input

บรรทัดเดียว: จำนวน `a` ซึ่งเป็นเลขฐานสอง ความยาวไม่เกิน `30` หลัก

## Output

แสดงค่าของจำนวนเต็ม `a` ในฐานสิบ
